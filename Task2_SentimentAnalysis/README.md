# Sentiment Analysis using NLP

## Overview

This project performs sentiment analysis on IMDb movie reviews using Natural Language Processing (NLP).

## Objective

Classify reviews as **Positive** or **Negative** using:

- TF-IDF Vectorization
- Logistic Regression

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- NLTK
- Jupyter Notebook

## Workflow

1. Load Dataset
2. Data Cleaning
3. Remove Stopwords
4. TF-IDF Vectorization
5. Train-Test Split
6. Logistic Regression
7. Model Evaluation
8. Custom Sentiment Prediction

## Output

- Accuracy Score
- Confusion Matrix
- Classification Report
- Sentiment Prediction for Custom Reviews
