# CNN Image Classification using TensorFlow

## Overview

This project builds a Convolutional Neural Network (CNN) to classify images from the Fashion MNIST dataset using TensorFlow and Keras.

## Dataset

Fashion MNIST (Automatically downloaded by TensorFlow)

Classes:
- T-shirt
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle Boot

## Technologies

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

## Workflow

1. Load Dataset
2. Normalize Images
3. Build CNN Model
4. Train Model
5. Evaluate Accuracy
6. Plot Accuracy & Loss
7. Confusion Matrix
8. Classification Report
9. Predict Images
10. Save Model

## Expected Accuracy

Approximately **90–93%** test accuracy.
