# Titanic Survival Prediction using Decision Tree

## Overview

This project uses a Decision Tree Classifier from Scikit-learn to predict whether a passenger survived the Titanic disaster based on passenger information.

## Dataset

- Titanic Dataset (Kaggle)
- File used: `train.csv`

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow

1. Data Loading
2. Data Exploration
3. Handling Missing Values
4. Encoding Categorical Variables
5. Feature Selection
6. Train-Test Split
7. Decision Tree Model Training
8. Model Evaluation

9. Feature Importance Analysis
10. Decision Tree Visualization

## Results

- Model Accuracy: **79%**
- Decision Tree successfully visualized.
- Classification report and confusion matrix generated.
