# Movie Recommendation System

## Overview

This project implements a Movie Recommendation System using Collaborative Filtering and Cosine Similarity.

## Dataset

MovieLens Latest Small Dataset

Files Used:

- movies.csv
- ratings.csv

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Workflow

1. Load Dataset
2. Merge Movie and Rating Data
3. Create User-Movie Matrix
4. Compute Cosine Similarity
5. Recommend Similar Movies
6. Display Top Rated Movies
7. Visualize Rating Distribution
8. Analyze User Activity

## Features

- Recommend Top 10 Similar Movies
- Collaborative Filtering
- Cosine Similarity
- Ratings Visualization
- User Activity Analysis
